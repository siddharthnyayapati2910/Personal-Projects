# Hybrid Retail Recommendation System

## Overview

This project was completed as part of the ISB Advanced Management Programme in Business Analytics (AMPBA).

## Business Problem

Retail customers often forget to reorder products they regularly purchase. The objective was to build a recommendation system that predicts which products a customer is likely to have forgotten while placing a new order.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NetworkX

## Machine Learning Techniques

- Matrix Factorization (SVD)
- Association Rules
- Personalized PageRank
- Recommendation Systems

## Key Outcomes

- Built a hybrid recommendation engine by combining multiple recommendation techniques.
- Evaluated model performance using Recall@5.
- Generated personalized product recommendations for customers.
- Compared different recommendation approaches to improve prediction accuracy.
