# FMCG Company Segmentation using Unsupervised Learning

## Overview
This project was completed as part of the ISB Advanced Management Programme in Business Analytics (AMPBA).

## Business Problem
Financial ratios alone do not always provide a clear understanding of how companies compare structurally. The objective was to group FMCG companies with similar financial characteristics using unsupervised machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Techniques
- K-Means Clustering
- Agglomerative Clustering
- PCA
- Isolation Forest

## Key Outcomes
- Clustered 50 FMCG companies based on financial ratios.
- Validated clusters using silhouette score and hierarchical clustering.
- Identified structural outliers using Isolation Forest.
- Visualized clusters using PCA.
